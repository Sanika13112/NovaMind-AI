# 🦄 NovaMind AI

NovaMind AI is a futuristic startup success prediction platform built using Machine Learning and Streamlit.

## 🚀 Features

- Startup success prediction
- Unicorn potential analysis
- AI recommendations
- Interactive analytics dashboard
- Futuristic glassmorphism UI
- Investor reaction system

## 🛠 Tech Stack

- Python
- Streamlit
- Scikit-Learn
- Plotly
- Pandas
- NumPy
